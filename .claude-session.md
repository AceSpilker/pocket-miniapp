## 项目概况
前后端分离的微信小程序框架，带完整的权限管理系统。

## 启动方式
```bash
# 后端
cd pocket-miniapp/backend
source .venv/bin/activate
python main.py

# 前端：微信开发者工具导入 frontend 目录
```

## 后端 API 一览 (36个接口)
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/auth/register | 账号注册 |
| POST | /api/v1/auth/login | 账号登录 |
| POST | /api/v1/auth/wx-login | 微信登录 |
| POST | /api/v1/auth/refresh | 刷新Token |
| POST | /api/v1/auth/logout | 登出 |
| GET | /api/v1/auth/verify | 验证Token |
| GET | /api/v1/user/me | 获取用户信息 |
| PUT | /api/v1/user/me | 更新资料 |
| POST | /api/v1/user/avatar | 上传头像(base64) |
| PUT | /api/v1/user/password | 修改密码 |
| DELETE/POST | /api/v1/user/bind-wechat | 绑定/解绑微信 |
| **管理后台** | | |
| GET/POST | /api/v1/admin/users | 用户列表/创建 |
| GET/PUT/DELETE | /api/v1/admin/users/{id} | 用户详情/编辑/删除 |
| PUT | /api/v1/admin/users/{id}/roles | 分配角色 |
| PUT | /api/v1/admin/users/{id}/reset-password | 重置密码 |
| GET/POST | /api/v1/admin/roles | 角色列表/创建 |
| PUT/DELETE | /api/v1/admin/roles/{id} | 角色编辑/删除 |
| GET/POST | /api/v1/admin/menus | 菜单列表/创建 |
| PUT/DELETE | /api/v1/admin/menus/{id} | 菜单编辑/删除 |
| PUT | /api/v1/admin/roles/{id}/menus | 分配菜单给角色 |
| GET | /api/v1/admin/user/menus | 当前用户可见菜单 |
| GET | /api/v1/admin/user/permissions | 当前用户权限 |

## 数据库(MySQL)
- 库: `pocket_miniapp`
- 表: users, roles, user_roles, menus, role_menus
- admin角色: 所有9种权限
- user角色: user:read
- 所有表支持逻辑删除(deleted_at)、审计字段(created_by/updated_by)
